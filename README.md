# Foco — Dashboard de Gestión de Hábitos

SPA desarrollada en React para la gestión de hábitos y tareas diarias. Evaluación Sumativa 3.

## Funcionalidades

- **CRUD completo** de hábitos con persistencia en LocalStorage
- **Heatmap** visual de los últimos 28 días (estilo GitHub)
- **Estadísticas**: total completados, consistencia, mejor racha
- **Widget de cita motivacional** vía API externa (dummyjson)
- **Sanitización de inputs** y validación de datos
- **Zona horaria Chile** (America/Santiago)
- **16 íconos predefinidos** para personalizar hábitos
- **Notificaciones Toast** con feedback visual
- **Diseño responsivo** con Bootstrap 5
- **Tema magenta/amarillo** con animaciones CSS

## Scripts

```bash
npm start       # Inicia servidor de desarrollo
npm test        # Ejecuta tests
npm run build   # Build producción
```

## Tecnologías

- React 19
- Bootstrap 5 + Bootstrap Icons
- LocalStorage API
- fetch API
